# Support

Use GitHub issues for support after the repository is public.

Before filing an issue, include:

- Android version and device model.
- Verbally version.
- Install source: Play Store, F-Droid, GitHub release, or local debug build.
- The app where text insertion failed, if relevant.
- Whether the issue affects Accessibility, overlay permission, recording, provider
  setup, or text insertion.

Do not include API keys, private transcripts, audio recordings, screenshots with
private text, or signing material in public issues.
