Verbally 是 Android 14+ 的浮動聽寫 app。

在文字框旁點一下浮動按鈕，開始說話，確認錄音後，Verbally 會把轉錄與整理後的文字插入目前游標位置。

主要功能：

• Android 文字框旁的浮動聽寫按鈕
• 使用者自行提供 API Key
• OpenAI、Soniox、Groq、Deepgram 語音轉文字
• OpenAI 或 Gemini 文字整理
• 本機歷史保留設定
• 本機字典與片段
• 優先直接插入游標位置，必要時才退回剪貼簿

Verbally 使用 Android 輔助使用偵測可輸入文字框、顯示浮動聽寫按鈕，並把聽寫文字插入或確認到目前游標位置。開啟輔助使用設定前，app 會先顯示獨立揭露並要求使用者同意。

Verbally 沒有後端、帳號系統、分析 SDK、廣告 SDK、崩潰回報 SDK 或雲端同步。音訊是暫存的，會在成功、失敗或取消後刪除。AI provider 請求只會送到你用自己 API Key 選擇的服務。
