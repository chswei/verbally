## Requirements
### Requirement: Dictionary destination is searchable and addable
The system SHALL provide a Dictionary destination with a search field, empty state, and add action.

#### Scenario: Dictionary has no entries
- **WHEN** the user opens Dictionary before entries exist
- **THEN** the screen shows a search field
- **THEN** the screen explains that dictionary words will show there
- **THEN** the screen shows an add action

### Requirement: Snippets destination is searchable and addable
The system SHALL provide a Snippets destination with a search field, empty state, and add action.

#### Scenario: Snippets has no entries
- **WHEN** the user opens Snippets before entries exist
- **THEN** the screen shows a search field
- **THEN** the screen explains that snippets will show there
- **THEN** the screen shows an add action
